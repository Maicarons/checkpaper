## 简介

CheckPaper 是一个基于 AI 的论文验证智能体系统，能够自动检测学术论文中的各种问题，帮助研究者提高论文质量。

### 主要功能

- **格式检查**：检查论文格式、结构、目录对应关系
- **图表引用检查**：检测图片/表格是否在文中被显式引用
- **参考文献引用检查**：验证参考文献是否在文中被引用
- **数据来源验证**：验证论文中数据来源的真实性
- **数据处理验证**：验证论文数据处理的真实性
- **参考文献验证**：联网搜索验证参考文献的真实性

### 支持的文件格式

- PDF (.pdf)
- Word (.docx)
- LaTeX (.tex, .latex)

## 快速开始

### 方式一：在线使用

1. 访问 CheckPaper 网站
2. 点击"开始验证"按钮
3. 上传您的论文文件
4. 选择需要验证的项目
5. 等待验证完成
6. 查看并下载验证报告

### 方式二：本地部署

#### 环境要求

- Python 3.11+
- Node.js 18+
- Docker (可选)

1. **克隆项目**
```bash
git clone https://github.com/yourusername/checkpaper.git

```

2. **配置环境变量**

3. **启动后端**


## 功能说明

### 1. 格式检查

检查论文的格式规范性，包括：

- **标题层级**：检查标题编号是否连续、层级是否清晰
- **字体字号**：检查字体、字号是否一致
- **段落格式**：检查段落缩进、间距是否规范
- **页面布局**：检查页边距、页眉页脚是否符合要求
- **目录对应**：检查目录与正文是否对应

### 2. 图表引用检查

检查论文中的图片和表格：

- **引用完整性**：所有图表是否在正文中被引用
- **编号匹配**：引用编号是否与实际定义匹配
- **孤儿引用**：检测引用了不存在的图表
- **未引用图表**：检测定义了但未被引用的图表

### 3. 参考文献引用检查

检查参考文献的引用情况：

- **引用一致性**：正文引用与参考文献列表是否匹配
- **重复引用**：检测重复的引用
- **缺失引用**：检测缺失的引用
- **格式规范**：检查引用格式是否统一

### 4. 数据来源验证

验证论文中的数据来源：

- **来源标注**：数据来源是否明确标注
- **URL可访问性**：数据集URL是否可访问
- **内容匹配**：数据描述是否与实际内容匹配
- **造假嫌疑**：检测可能的数据造假

### 5. 数据处理验证

验证论文中的数据处理方法：

- **统计方法**：检查统计方法是否正确
- **样本量**：检查样本量是否充足
- **p值合理性**：验证p值、置信区间是否合理
- **数据一致性**：检查图表数据与正文报告是否一致

### 6. 参考文献验证

验证参考文献的真实性：

- **DOI验证**：验证DOI是否存在且有效
- **元数据匹配**：检查标题、作者、期刊信息是否匹配
- **期刊可信度**：检测虚假期刊或掠夺性期刊
- **联网搜索**：通过多个学术数据库验证文献

## 使用指南

### 上传论文

1. 点击"上传论文"按钮
2. 选择论文文件（支持拖拽上传）
3. 系统自动识别文件类型
4. 选择需要验证的项目
5. 点击"开始验证"

### 查看验证进度

上传后，系统会自动开始验证：

- **等待中**：任务已创建，等待执行
- **验证中**：AI 正在分析论文
- **已完成**：验证完成，可以查看报告
- **失败**：验证失败，请重试

### 查看验证报告

验证完成后，可以查看详细报告：

1. **验证总结**：显示总问题数、严重问题数等
2. **各项检查结果**：每项检查的详细结果
3. **问题列表**：发现的所有问题
4. **改进建议**：针对每个问题的改进建议

### 下载报告

支持多种格式下载：

- **Markdown**：纯文本格式，便于编辑
- **HTML**：网页格式，便于分享
- **PDF**：打印格式，便于存档

## 常见问题

### Q: 支持哪些文件格式？

A: 目前支持 PDF、Word (.docx)、LaTeX (.tex, .latex) 格式。

### Q: 文件大小有限制吗？

A: 单个文件最大支持 50MB。

### Q: 验证需要多长时间？

A: 验证时间取决于论文长度和选择的验证项目，通常在 1-5 分钟内完成。

### Q: 验证结果准确吗？

A: 验证结果基于 AI 分析和多个学术数据库比对，具有较高的准确性。但建议人工复核重要发现。

### Q: 如何获取 OpenAI API Key？

A: 访问 https://platform.openai.com/ 注册账号并创建 API Key。

### Q: 可以批量验证吗？

A: 目前支持单篇验证，批量验证功能正在开发中。

### Q: 数据安全有保障吗？

A: 是的，所有上传的文件都会在验证完成后自动删除，不会保存或用于其他用途。

### Q: 验证失败怎么办？

A: 验证失败可能是以下原因：
- 文件格式不支持
- 文件损坏
- 网络问题
- API 配额用尽

请检查文件格式，或稍后重试。
